Getting Started

This example demonstrates basic voice command functionality for the ManVue application. Click the "Start Voice Recognition" button below and try speaking one of the example commands.

Navigation Commands

Try these commands to navigate the application:

  • "Go home" - Navigate to the main page
  • "Show products" - View the product catalog
  • "Open cart" - View shopping cart
  • "View profile" - Open user profile

Search Commands

Use these commands to search for products:

  • "Search for red dress" - Find red dresses
  • "Show me blue jeans" - Find blue jeans
  • "Find sneakers" - Search for sneakers
  • "Show shirts" - Browse shirt category

Product Actions

These commands work when viewing a product:

  • "Add to cart" - Add current product to cart
  • "Show details" - View product information
  • "Compare products" - Add to comparison

System Commands

General system commands:

  • "Help" - Show available commands
  • "Stop listening" - Disable voice recognition
