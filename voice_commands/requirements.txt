# Installation Instructions:
# 
# 1. Basic installation:
#    pip install -r requirements.txt
#
# 2. For development:
#    pip install -r requirements.txt
#    pip install -e .
#
# 3. For production (minimal):
#    pip install speechrecognition pyttsx3 fastapi uvicorn websockets numpy
#
# Platform-specific notes:
# 
# Windows:
#   - May need to install Microsoft Visual C++ Build Tools
#   - PyAudio installation might require: pip install pipwin && pipwin install pyaudio
#
# macOS:
#   - May need to install PortAudio: brew install portaudio
#   - Ensure Xcode command line tools are installed
#
# Linux (Ubuntu/Debian):
#   - sudo apt-get install python3-dev python3-pip
#   - sudo apt-get install portaudio19-dev
#   - sudo apt-get install flac  # for better speech recognition
#
# Docker requirements:
#   - Base image should include audio libraries
#   - May need pulseaudio for container audio support
