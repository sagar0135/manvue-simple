KEY DATA FLOWS:
===============

1. USER UPLOAD IMAGE → Visual Search Service → CLIP Embedding → MongoDB Similarity Search → Ranked Results
2. VOICE COMMAND → Text Embedding → Product Similarity Search → Voice Response
3. CHATBOT QUERY → Enhanced ML Service → Intelligent Product Recommendations → Natural Language Response
4. COLAB PROCESSING → Batch Image Conversion → GridFS Storage → Embedding Generation → Database Update
5. REAL-TIME SEARCH → Embedding Comparison → MongoDB Query → Ranked Product Results
